# Formulário Inspeção de Infraestrutura Física de CME
Todos os inputs do [Formulário de Inspeção de Infraestrutura Física](form-inspecao-estrutura-fisica.md) mais os itens abaixo
- O serviço é próprio?
- O serviço é contratado?
- Fluxo adequado de materiais e funcionários?
- Possui sala de utilidades/expurgo exclusivo para lavagem de material?
- Qual tipo de limpeza utilizado? (manual, ultrassônica, automatizada)
- Normatização dos procedimentos internos
- Autoclave de barreira?
- Mapa controle de temperatura das autoclaves?
- Controle de qualidade dos procedimentos de esterilização (bowie dick)?
- Controle de qualidade dos procedimentos de esterilização (Teste biológico semanal)?
- Autoclave à vapor?
- Possui serviço de Oxetil?
- Possui área reservada para registro e arquivo de fichas? 
- Possui sala administrativa?
- Possui paredes com bordas arredondadas e tinta que permite lavagem?
- Possui armazenamento e acondicionamento de instrumental cirúrgico, identificado e dentro do prazo de validade de esterilização?
