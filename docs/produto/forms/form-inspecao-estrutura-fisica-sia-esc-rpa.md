# Formulário Inspeção de Infraestrutura Física de SIA, Esc e RPA
Todos os inputs do [Formulário de Inspeção de Infraestrutura Física](form-inspecao-estrutura-fisica.md) mais os itens abaixo
- Possui materiais e equipamentos básicos para anestesia?
- Possui carro de emergência com materiais adaptados para uso adulto, pediátrico e em neonatologia?
- Possui área de escovação com condições de lavagem e anti-sepsia cirúrgica das mãos?
- A sala cirúrgica possui porta com visor?
- Possui sistema de assistência respiratória por: ( ) Sistema de parede ( ) Bala/torpedo
- A sala de RPA possui área física com localização de fácil acesso?
- Possui camas com grade?
- Possui sistema de monitorização multiparamétrica?
- Possui sistema de assistência respiratória na RPA?
- Possui posto de enfermagem que atende a RPA?
